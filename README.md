# RxChat
A reactive WebSocket-based chat in JavaScript using RxJS:
![video](https://user-images.githubusercontent.com/62397363/88463935-d8822900-ceb6-11ea-9dab-99820a875d0e.gif)

## Prerequisites:
- Node.js 12.18.4

## Build and run:
### Node.js:
- `npm install`
- `npm start`
### live-server:
- `npx live-server`
