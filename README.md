# RxChat
A websocket-based chat in JavaScript using RxJS:
![video](https://user-images.githubusercontent.com/62397363/88463935-d8822900-ceb6-11ea-9dab-99820a875d0e.gif)

## Build and run:
```
 npm install
```
```
 npm start
```
```
 npx http-server -o
```
